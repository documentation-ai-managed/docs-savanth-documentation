<Update label="2024-08-10" description="v1.0.0" tags={["feature", "breaking"]}>

## New Features

- Full YAML frontmatter validation with no blank lines

## Breaking Changes

- Removed H1 support; always start with H2 headings

</Update>

## Upgrade Guide

Follow these steps to upgrade Savanth Documentation to the latest version.

<Steps>
  <Step title="Check Current Version" icon="package">
    Run the version check command.

    ```bash
    savanth --version
    ```
  </Step>
  <Step title="Update Package" icon="upload">
    Use your package manager.

    <CodeGroup tabs="npm,yarn,pnpm">
      ```bash
      npm update savanth@latest
      ```
      ```bash
      yarn upgrade savanth@latest
      ```
      ```bash
      pnpm update savanth@latest
      ```
    </CodeGroup>
  </Step>
  <Step title="Rebuild Docs" icon="refresh-cw">
    Rebuild your documentation site.

    ```bash
    savanth build
    savanth serve
    ```
  </Step>
</Steps>

<Callout kind="tip">
  After upgrading, test your pages for parsing errors, especially nested components and special characters.
</Callout>
